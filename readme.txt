Intègre la passerelle de paiement Kkiapay à GiveWP.

== Description ==
Ce plugin ajoute Kkiapay comme option de paiement dans GiveWP, permettant d'accepter les paiements par carte bancaire et mobile money.

== Installation ==
1. Téléchargez le plugin
2. Extrayez le fichier ZIP
3. Uploader le dossier dans /wp-content/plugins/
4. Activez le plugin dans WordPress
5. Configurez les clés API dans Donations → Paramètres → Passerelles de paiement → Kkiapay

== Configuration ==
1. Obtenez vos clés API sur https://kkiapay.me
2. Entrez les clés dans les paramètres du plugin
3. Configurez l'URL de webhook dans votre dashboard Kkiapay
